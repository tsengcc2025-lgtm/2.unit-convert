單位換算助手 — 輸入 exit 結束

測試範例：
  • 25 度 C 是華氏幾度？
  • 10 公里等於幾英里？
  • 70 公斤是幾磅？

✔ 請輸入你的問題： 25度C是華氏幾度

[呼叫 tool] convert_unit({"value":25,"from_unit":"C","to_unit":"F"})

25°C = 77°F

✔ 請輸入你的問題： 10公里等於幾英里

[呼叫 tool] convert_unit({"value":10,"from_unit":"km","to_unit":"mile"})

10公里 = 6.21371英里

✔ 請輸入你的問題： 70公斤是幾磅

[呼叫 tool] convert_unit({"value":70,"from_unit":"kg","to_unit":"lb"})

70公斤 = 154.3234磅
